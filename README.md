# skills-by-yigitkonur

All my AI agent skills in one place.

## Install

```bash
# Install all skills
npx skills add yigitkonur/skills-by-yigitkonur

# Install a specific skill
npx skills add yigitkonur/skills-by-yigitkonur/design-soul-saas
npx skills add yigitkonur/skills-by-yigitkonur/devin-review-init
npx skills add yigitkonur/skills-by-yigitkonur/greptile-config
npx skills add yigitkonur/skills-by-yigitkonur/mcp-cli
npx skills add yigitkonur/skills-by-yigitkonur/mcp-server-tester
npx skills add yigitkonur/skills-by-yigitkonur/snapshot-to-nextjs
npx skills add yigitkonur/skills-by-yigitkonur/tauri-devtools
```

## Skills

| Skill | Description |
|---|---|
| **[design-soul-saas](skills/design-soul-saas/)** | Forensic visual DNA extraction for SaaS dashboards. Extracts every design token, component pattern, animation, and responsive behavior into structured documentation that downstream builders can implement pixel-perfect. |
| **[devin-review-init](skills/devin-review-init/)** | Generate production-ready `REVIEW.md` and `AGENTS.md` files for Devin Review's AI code review system (Bug Catcher). Analyzes the repo and produces tailored review instructions. |
| **[greptile-config](skills/greptile-config/)** | Generate production-ready Greptile AI code review configuration. Analyzes repo structure, tech stack, and conventions to produce tailored `.greptile/` config files with semantic rules. |
| **[mcp-cli](skills/mcp-cli/)** | Guide for using the `mcp` CLI tool to inspect, test, and debug MCP servers interactively. Covers connection, tool listing, tool execution, and output chaining. |
| **[mcp-server-tester](skills/mcp-server-tester/)** | Comprehensive MCP server testing skill. Covers basic protocol compliance, tool validation, error handling, and LLM-driven behavioral testing via MCP Inspector. |
| **[snapshot-to-nextjs](skills/snapshot-to-nextjs/)** | Convert saved HTML snapshots (browser "Save As", wget, HTTrack) into pixel-perfect Next.js App Router projects with zero external dependencies. Forensic CSS parsing, self-hosted assets, Tailwind config grounded on real extracted values. |
| **[tauri-devtools](skills/tauri-devtools/)** | Debug Tauri v2 desktop apps using the Tauri DevTools instrumentation system. Covers IPC span inspection, state tracking, log correlation, and performance analysis. |
