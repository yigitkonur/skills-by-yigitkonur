# orchestrate-projects-by-jean

Supervise existing coding-agent projects in Jean through MCP and Computer Use, including inventory, recovery, bounded monitoring, completion verification, or worktree closure.

**Category:** orchestration

## Install

**As a plugin (easy install / uninstall via `/plugin`):**

```
/plugin marketplace add yigitkonur/skills-by-yigitkonur
/plugin install orchestrate-projects-by-jean@yigitkonur
```

**Or with the `skills` CLI — this skill only:**

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/orchestrate-projects-by-jean
```

**Or the full pack:**

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
