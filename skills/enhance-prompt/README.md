# enhance-prompt

Rewriting a task prompt for an LLM or coding agent to improve execution quality without changing skill instructions or doing the task.

**Category:** productivity

## Install

Install this skill individually:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/enhance-prompt
```

Or install the full pack:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
