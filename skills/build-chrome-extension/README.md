# build-chrome-extension

Building or debugging a Chrome MV3 extension with service workers, content scripts, popup/sidepanel, messaging, storage, permissions, testing, or Web Store publishing.

**Category:** development

## Install

**As a plugin (easy install / uninstall via `/plugin`):**

```
/plugin marketplace add yigitkonur/skills-by-yigitkonur
/plugin install build-chrome-extension@yigitkonur
```

**Or with the `skills` CLI — this skill only:**

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/build-chrome-extension
```

**Or the full pack:**

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
