# run-codex-exec

Redirecting deprecated codex exec orchestration installs to use-codex exec mode.

**Category:** orchestration

## Install

Install this compatibility stub individually:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/run-codex-exec
```

Or install the full pack:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
