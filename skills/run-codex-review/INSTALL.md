# run-codex-review

Redirecting deprecated codex review orchestration installs to run-codex-2 review mode.

**Category:** orchestration

## Install

Install this compatibility stub individually:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/run-codex-review
```

Or install the full pack:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
