# audit-ui-and-save-files

Auditing a running web app's UI across pages and viewports, saving per-bug findings to a dated tree, and dispatching approval-gated fix subagents.

**Category:** productivity

## Install

**As a plugin (easy install / uninstall via `/plugin`):**

```
/plugin marketplace add yigitkonur/skills-by-yigitkonur
/plugin install audit-ui-and-save-files@yigitkonur
```

**Or with the `skills` CLI — this skill only:**

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/audit-ui-and-save-files
```

**Or the full pack:**

