# use-skill-dl-util

Searching, discovering, and downloading AI coding skills using skill-dl with Serper-powered search and batch downloads.

**Category:** productivity

## Install

Install this skill individually:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur/skills/use-skill-dl-util
```

Or install the full pack:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur
```
